# RICARDOOOOOO

### Installation and running


```sh
$ npm install
$ npm run serve
```
### Useage
So basically just go to http://localhost:1337/ and then add in the query string you want to add.
eg: http://localhost:1337/?page=1
eg: http://localhost:1337/?page=2
etc...
This will scrape it, render it, and from there you can parse the HTML for the data
